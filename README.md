# hamburger-shop
ハンバーガーショップのWebアプリ
```
【製作期間】	約3時間
【使用言語】	PHP,HTML/CSS,Javascript
【ジャンル】	Webアプリケーション
【概要】		ハンバーガーショップの注文を体感できるWebアプリケーション
【操作方法】	マウスクリック	
【書いた範囲】	コピー＆ペーストしたものもあるが全部
【制作動機】	授業で作ったものの機能が少なすぎたので多少追加してみたかった。
【頑張った所】	特に「ご一緒にポテトはいかがですか？」システムを頑張った。
【面白いところ】	ポテトを注文していようが関係なしにポテトを勧められるところ。極稀に人間でもいるらしい。
```
